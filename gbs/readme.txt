The controls are as follows.

A	- Play selected song
B	- Play selected sfx
Right	- Increase selected song
Left	- Decrease selected song
Up	- Increase selected sfx
Down	- Decrease selected sfx
Select	- Switch playmode between Infinite & automatic advance
Start	- Stop or resume current song.

The GBS file can be played in any gbs player.
The GBS is organized with all of the songs playing first, then, if you loaded it without
the .m3u, ALL of the SFX.  

The .m3u file, used with a compatible gbs player, gives song title/timing information

The rom may be used with any gameboy emulator or a flash cart.
The recommended emulator, for sound accuracy, is higan ( http://byuu.org/ )
